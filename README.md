# 👩‍💼 HR Employee Distribution Report

## 📌 Description
This project focuses on analyzing HR data to understand employee distribution across different factors such as department, location, gender, and job roles. The dataset was cleaned and analyzed using SQL, and insights were visualized through an interactive Power BI dashboard.

## 🎯 Objective
To explore HR data and generate insights that help in understanding workforce distribution and key HR metrics.

## 🚀 Features
- Employee distribution by department
- Gender-wise employee analysis
- Location-based workforce distribution
- Job role and employment type insights
- Data cleaning and preprocessing of HR dataset
- SQL-based analysis for business questions

## 🛠️ Tech Stack
- SQL (data cleaning & analysis)
- Power BI (dashboard & visualization)
- CSV Dataset (HR data)

## 📂 Project Structure
-  Raw dataset
-  Data cleaning and analysis SQL queries
-  Power BI dashboard file

## ▶️ How to Run
1. Open the CSV file in your SQL environment
2. Run SQL queries for data cleaning and analysis
3. Import cleaned data into Power BI
4. Open the file to view the dashboard

## 📊 Key Insights
- Identified employee distribution across departments
- Analyzed gender diversity in the organization
- Observed workforce concentration by location
- Highlighted key HR trends using SQL queries
